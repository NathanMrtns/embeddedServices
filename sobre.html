O Android Tech Center tem como objetivo a formação de recursos humanos em tecnologias de softwares para dispositivos portáteis, direcionando a capacitação para um desenvolvimento de software profissional. A plataforma alvo deste projeto foi Android, devido à abrangência de dispositivos compatíveis e o amplo suporte de ferramentas para o desenvolvimento de aplicações.
